## Access heroku cloud log
1. Install [heroku CLI](https://devcenter.heroku.com/articles/heroku-cli)
2. Login with CLI: ```heroku login```
3. View logs: ```heroku logs -a bco-cloud``` or ```heroku logs -a bco-cloud --tail```
